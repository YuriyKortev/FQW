Тестовый Qt проект, демонстрирующий работу детектора трещин на поверхности РБМК.

cracks_detector.cpp
cracks_detector.h
- заголовочный и исходный файлы класса, реализующего детекцию трещин.

В конструктор класса необходимо передать путь к файлу с весами и архитектурой модели, сохраненному в формате tensorflow.

mainwindow.cpp
mainwindow.h
- заголовочный и исходный файлы класса главного окна.
В исходном файле реализована демонстрация работы детектора трещин.

main.cpp
- исходный файл с точкой входа в программу.

mainwindow.ui
- разметка главного окна.

cracks_detector.pro
- файл проекта.

frozen_graph.pb
- бинарный файл с моделью Unet, сохраненной в формате tensorflow.

test_images
- тестовые изображения.

Для класса cracks_detector реализовано 4 публичных метода:

detect_single_frame
- принимает путь к исходному изображению и путь, где будет сохранен результат с выделенными дефектами.
detect_dir_of_frames
- принимает путь к каталогу с исходными изображениями и путь к каталогу, где будет сохранен результат, для каждого изображения.
process_image
- принимает cv::Mat объект, в котором хранится изображение и сохраняет результат по ссылке на второй аргумент с типом cv::Mat.
detect_from_cam
- принимает номер устройства, считывает и обрабатывает с него изображения, результат выводит на экран, до тех пор, пока не нажата клавиша esc.
get_cracks
- принимает вектор путей к кадрам, которые должны быть отсортированы по отметке высоты, возвращает вектор с информацией(координаты верхней и нижней точки, ширина) для каждой найденной трещины.
